# 🌍 Geo-Aid

**Geo-Aid** is a real-time, location-based disaster awareness and community response platform designed to help people **prepare, respond, and act** during natural disasters.

It combines **live environmental updates, disaster-specific precautions, citizen reporting, and volunteer/donation coordination** into a single, accessible platform.

---

## 🚀 Features

- 📡 **Live Disaster Updates**  
  Real-time news and environmental alerts using verified APIs

- 🧭 **Location-Based Visualization**  
  Interactive maps (OpenStreetMap + Leaflet) to view affected areas and shelters

- 📸 **Citizen Reporting**  
  Upload geotagged images and updates from disaster zones

- 📘 **Disaster Awareness & Precautions**  
  Guidelines for floods, earthquakes, cyclones, and more

- 🤝 **Volunteer System**  
  Users can register as volunteers (remote/on-site)

- 💰 **Donation Module**  
  Structured donation system for disaster relief

- 🔐 **Authentication System**  
  Separate login for individuals and organizations (NGOs)

## 🏗️ System Architecture

Geo-Aid follows a **3-layer architecture**:

1. **Client Layer (Frontend)**  
   Handles user interaction and UI rendering

2. **Application Layer (Flask Backend)**  
   Processes logic, API integration, and user requests

3. **Data Layer (Database + APIs)**  
   Stores structured data and fetches real-time information

---

## ⚡ How It Works

1. Users receive **alerts and environmental updates**
2. They can view **affected areas on the map**
3. Users can **upload real-time reports**
4. Others can **volunteer or donate**
5. NGOs can **coordinate relief efforts**

---

## 🎯 Problem It Solves

- Fragmented disaster information
- Misinformation on social media

- Poor coordination between citizens and relief organizations
